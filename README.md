# 📊 Data Structures Visualizer

<https://athulkrishnapulari-sudo.github.io/DS-Visualizer/>

An interactive web-based **Data Structures Visualizer** that helps users understand and learn fundamental data structures through visual representation.

Currently supports:
- 📚 Stack (Push, Pop, Peek)
- 🔄 Queue (Enqueue, Dequeue, Peek)
- 🔁 Circular Queue
- 🔗 Linked List Operations

Built using **HTML, CSS, and JavaScript**, this project provides a simple way to visualize how data structures work internally and makes learning DSA concepts easier.
